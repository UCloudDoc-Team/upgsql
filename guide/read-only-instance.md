# 从库管理
云数据库 UDB PostgreSQL 支持为主库实例创建从库，用于分担主库的读负载，提升整体服务性能。在支持多可用区的地域中，用户可以选择将备库部署在与主库不同的可用区，从而实现可用区级别的容灾。当创建从库时，需要逐实例创建，目前不支持批量创建。



## 创建从库
**前置条件**

已创建主库实例且实例处于运行中状态

**操作步骤**

1. 访问 [UDB PostgreSQL 实例列表](https://console.ucloud.cn/postgresql/postgresql)，找到目标实例，点击操作列中的“创建从库”按钮。

![](/images/1763982130315-0f7bb5cb-664c-43ae-8325-3f3984e82ec6.png)

2. 选择选择所需配置及计费方式，点击“立即购买”。配

![](/images/1763982201789-66027ab8-aa66-4c22-a22f-83c715b71edd.png)

3. 确认订单，并支付。

![](/images/1763982303893-7a645d16-9e6e-4e02-8f79-c6d12567b506.png)



## 提升为主库
**前置条件**

已创建从库实例且实例处于运行中状态。

**操作步骤**

1. 访问 [UDB PostgreSQL 实例列表](https://console.ucloud.cn/postgresql/postgresql)，选择需要提升为主库的目标从库实例，点击操作项中“提升为主库”按钮。

![](/images/1764038328472-8a8345d7-e38d-463a-824d-0978dd83750f.png)

2. 弹窗“确认”，执行成功后实例的数据库类型从“从库版”变更为“普通版”或“高可用版”。

![](/images/1764038391922-147598ab-b29e-4a74-a1a0-fdf1b069424e.png)

![](/images/1764038443600-8802f951-fc5d-4901-9ce3-db98976693d4.png)



## 更改配置
**前置条件**

已创建实例

**操作步骤**

1. 访问 [UDB PostgreSQL 实例列表](https://console.ucloud.cn/postgresql/postgresql)，选择需要更改配置的目标实例，点击操作列中“更改配置”按钮。

![](/images/1764039384926-b936f6b5-4e0f-4279-83e5-3567db0dc0a3.png)

2. 抽屉页修改配置，支持对CPU、计算规格、硬盘容量进行修改，并确认费用。 

> 注意
>
> + 对于降配场景，将在新规格计费的基础上，减去与原规格之间的费用差额，具体计费参考文档 [计费指南](/upgsql/price/bill) ；
> + 对于升配场景，则需在原规格费用基础上补足新增部分的费用。
>

![](/images/1764039466183-b44680df-fc2f-4885-a99b-0671aebddfd8.png)

3. 确认订单。

![](/images/1764039918833-2f6fe688-a6b3-4b1e-a83f-d059763f12e5.png)
