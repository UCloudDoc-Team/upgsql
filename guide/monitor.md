# 监控管理
云数据库 UDB PostgreSQL 在控制台为用户提供多项监控视图，包括：磁盘使用率、CPU使用率、内存大小、连接数、OpQPS、内存使用率、慢查询、存活探测失败次数等多项监控，并支持设置监控告警模板。

## 查看监控
访问控制台 [UDB PostgreSQL 实例列表](https://console.ucloud.cn/postgresql/postgresql) 进入目标实例详情页，在“概览”页面即可查看当前实例的全部监控信息

## 告警模板
### 绑定/解绑告警模板
云数据库 UDB MySQL 支持通过控制台配置告警模板，用于管理实例的告警规则与触发条件。系统默认提供一套预置告警模板，用户可根据需要选择并绑定多个自定义告警模板，以实现更灵活的告警策略管理。

**操作步骤**

1. 登录控制台 [UDB PostgreSQL 实例列表](https://console.ucloud.cn/postgresql/postgresql) 进入目标实例详情页；
2. “预览”标签页，左侧“基本信息”模板，点击“告警模板”行的编辑图标，打开告警配置窗口；
3. 在弹窗中选择需绑定的告警模板，可执行：
    - **绑定告警模板**
    - **解绑告警模板**
4. 点击 **“确定”** 保存配置。

![](/images/1764052937046-5a1a7bee-e4b9-47d3-9979-368f6b46f56c.png)
